HTML Escape Characters

HTML uses special codes called escape characters to display reserved symbols.

Common Escape Characters:

  • < > → Display < or > symbols instead of HTML tags
  • & → Display the ampersand (&) symbol
  • " → Display double quotes (")
  • ' → Display single quotes (')
  • © → Display © (Copyright)
  • ™ → Display ™ (Trademark)
  •   → Non-breaking space (keeps extra spaces)

Examples:

<h1>This is a heading</h1>

<p>This is a paragraph</p>

5 > 3 → 5 is greater than 3

2 < 10 → 2 is less than 10

Tom & Jerry → Ampersand symbol
