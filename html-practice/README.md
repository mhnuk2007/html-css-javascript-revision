# HTML Practice

Hands-on practice exercises for learning HTML fundamentals.

## Topics Covered

### 1. Basic HTML
- Document structure (`<!DOCTYPE>`, `<html>`, `<head>`, `<body>`)
- Headings (`<h1>` - `<h6>`), paragraphs (`<p>`)
- Comments and escape characters
- Inline formatting (`<strong>`, `<em>`, `<span>`, etc.)

### 2. Semantic Structure
- Semantic tags: `<header>`, `<main>`, `<section>`, `<article>`, `<aside>`, `<footer>`
- Proper page organization
- Accessibility best practices (aria-label, semantic markup)

### 3. Text & Links
- Hyperlinks (`<a>`) and navigation menus
- URL examples and relative/absolute paths
- Blockquotes (`<blockquote>`, `<q>`)
- Time (`<time>`) and date elements

### 4. Tables
- Basic table structure (`<table>`, `<tr>`, `<td>`, `<th>`)
- Table headers, captions, and sections (`<thead>`, `<tbody>`, `<tfoot>`)
- Row and column spans (`colspan`, `rowspan`)

### 5. Images
- Image attributes (`src`, `alt`, `width`, `height`, `loading`)
- Responsive images using CSS
- Responsive images using `<picture>` and `<source>`
- Pixel density handling (Retina/2x displays)

### 6. Media
- Audio element (`<audio>`) and attributes
- Video element (`<video>`) and attributes
- Media attributes: `controls`, `poster`, `autoplay`, `loop`, `muted`
- Embedding via `<iframe>` (YouTube, external content)

### 7. Forms
- Form structure (`<form>`, `<input>`, `<label>`, `<button>`)
- Input types: text, email, password, number, date, radio, checkbox
- Registration form with validation attributes
- Submit handling and form methods

## Running the Exercises

1. Open `html-practice/index.html` in a browser
2. Click any link to view that specific exercise
3. Modify the code to experiment and learn
